# HFT

Project Topic: 

We will be using Interactive Brokers (IBKR) to research High Frequency Trading (HFT) alpha signals on level 2 (L2) tick market data for US Equity Options. We will then use our signals to create a portfolio and forward test them on a live paper trading account in IBKR.

Project Timeline: 

2/11: Submit Project Proposal
2/15: Finish script to collect and pull tick data
2/16 - 2/20: Collect 1 week of tick data
4/1: Finish Signal Research with Backtesting
4/12: Finish Coding Framework for the Live Paper Trading with the Research Signals
4/13 - 4/24: Collect 2 Weeks of Live Paper Trading Results
5/1: Final Report

Project Goals: 

Successfully find and build profitable HFT strategies with a low drawdown and high sharpe ratio for US Equity Options

Data Source: 

IBKR US Equity and Options level 2 market data for common stocks (i.e. AAPL, TSLA, MSFT, ... etc)

Data Collection Method:

We will use IBKR Market Data API to collect L2 tick market data
