# HFT

Project Topic: 

We will be using Interactive Brokers (IBKR) to research High Frequency Trading (HFT) alpha signals on level 2 (L2) market data for US Equity Options. We will then use our signals to create a portfolio and forward test them on a live paper trading account in IBKR.
